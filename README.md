# 概要

本リポジトリでは、下記の機能を提供します。

| 機能名            | 機能概要                                                                                                                    |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------- |
| Shapeファイル正規化 | Shapeファイルのジオメトリ検証・修正、座標系変換（EPSG）、カラムマッピングを行います。複数ファイルの一括処理に対応しております。 |

# フォルダ構成

```
└── shapefile-normalizer-tool
    ├── src                             // ソースコード
    │   └── shapefile_normalizer_gui.py // メインアプリケーション
    ├── config                          // 設定ファイル格納ディレクトリ
    │   └── config.json                 // 設定ファイル
    ├── scripts                         // ビルドスクリプト
    │   ├── ShapefileNormalizerTool.spec // PyInstallerスペックファイル
    │   ├── build_cx_msi.bat            // MSIビルドスクリプト
    │   ├── build_exe.bat               // EXEビルドスクリプト
    │   └── setup_cx_only.py            // cx_Freezeセットアップスクリプト
    ├── builds                          // ビルド成果物格納ディレクトリ
    ├── requirements.txt                // 依存ライブラリ定義
    ├── ShapefileNormalizerTool.exe     // 実行ファイル（配布用）
    ├── ShapefileNormalizerTool-1.0.0-win64.msi  // インストーラー（配布用）
    └── README.md                       // 本ファイル
```

# リポジトリ利用方法

必要なソフトウェアやサービスの詳細は[前提条件](#前提条件)をご参照ください。

## 方法1: 配布済みEXE/MSIを使用する場合

1. 配布済みの実行ファイルを使用します。
   - `ShapefileNormalizerTool.exe`: ダブルクリックで起動（Pythonインストール不要）
   - `ShapefileNormalizerTool-1.0.0-win64.msi`: インストーラーを実行してインストール

## 方法2: Pythonスクリプトを直接実行する場合（開発・テスト用）

1. 依存ライブラリをインストールします。

   ```bash
   pip install -r requirements.txt
   ```

2. GUIアプリケーションを起動します。

   ```bash
   cd src
   python shapefile_normalizer_gui.py
   ```

## 方法3: EXE/MSIをビルドする場合

1. ビルドスクリプトを実行します。
   | ビルドスクリプト | 概要 |
   | ------------------------ | -------------------------------------------------------------------------------- |
   | scripts\build_exe.bat | PyInstallerで単一EXEファイルをビルド（依存ライブラリは自動インストールされます） |
   | scripts\build_cx_msi.bat | cx_FreezeでMSIインストーラーをビルド（依存ライブラリは自動インストールされます） |
2. ビルド成果物は `builds\dist\` ディレクトリに出力されます。

   **注意**: MSIビルド時にconfig.jsonを含めたい場合は、ビルド前に `config\config.json` に設定を記述してください。ビルド時に自動的にMSIに内包されます。

## アプリケーションの使用方法

3. 各タブの操作方法
   | 手順           | 内容                       | 詳細                                                                                                                                                                                           |
   | -------------- | -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | 実行タブ       | 入力ファイル選択と処理実行 | .shpまたは.zipファイルを複数選択し、出力先ディレクトリを指定します。各ファイルのマッピング定義、EPSG、文字コードを設定後、「実行」ボタンをクリックします。                                     |
   | 設定タブ       | デフォルト設定             | バッファ距離（最小辺の長さチェック）、デフォルトの変換元EPSG、変換先EPSG、文字コードを設定します。設定の保存・読み込みが可能です。                                                             |
   | カラム設定タブ | カラムマッピング定義       | サンプルファイルから入力カラム情報を読み込み、出力カラムに対するマッピング方法（カラム代入、四則演算、カラム抽出、固定値、ファイル名、ランダム値、シーケンス値、条件分岐、ファイル名分岐等）を設定します。 |

4. カラムマッピング方式一覧
   | マッピング方式     | 概要                                           |
   | ------------------| ---------------------------------------------- |
   | カラム代入 ※       | 入力カラムをそのまま出力カラムに代入           |
   | カラム四則演算 ※   | 入力カラムに対して四則演算を実行               |
   | 複数カラム四則演算 ※ | 複数の入力カラム間で四則演算を実行             |
   | 複数カラム抽出  ※  | 複数の入力カラム間で最大値・最小値を抽出   　　|
   | 固定値             | 固定値を設定                                   |
   | ファイル名         | インポートしたファイル名（.shp）を自動設定     |
   | ランダム値         | 指定範囲のランダム値を生成                     |
   | シーケンス値       | 連番を生成                                     |
   | 条件分岐           | 入力カラムの値に応じて出力値を分岐             |
   | ファイル名分岐     | ファイル名に応じて出力値を分岐（拡張子省略可） |

   **※ 補完機能**: 指定した入力カラムのNULL値・空文字列を補完
   - **固定値**: 固定値を代入
   - **カラム条件分岐**: 指定した他カラムの値をもとに出力値を代入
   - **ファイル名分岐**: インポートしたファイル名をもとに出力値を代入
