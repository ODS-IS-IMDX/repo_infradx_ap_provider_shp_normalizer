# 概要

本リポジトリでは、下記の機能を提供します。

| 機能名            | 機能概要                                                                                                                    |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------- |
| 初期設定ツール | データ変換ツールの実行時に必要な設定ファイルの作成を行います |
| ジオメトリ検証・修正 | 不正なジオメトリを検出し自動修復します                         |
| 座標系変換           | 任意のEPSGコード間で座標変換を行います                         |
| カラムマッピング     | 出力カラムの生成ルールを柔軟に定義できます                     |
| 一括処理             | 複数の `.shp` / `.zip` ファイルをまとめて処理できます          |
| 設定保存             | マッピング定義や各種設定をJSONファイルで保存・読み込みできます |

# リポジトリ利用方法
➀初期設定ツールの利用方法は[マニュアル](https://github.com/ODS-IS-IMDX/repo_infradx_ap_provider_shp_normalizer/tree/main/docs)をご参照ください。

➁データ変換ツールの利用方法
必要なソフトウェアやサービスの詳細は[前提条件](#前提条件)をご参照ください。

## 方法1: 配布済みEXEを使用する場合

1. 配布済みの実行ファイルを使用します。
   - `ShapefileNormalizerTool.exe`: ダブルクリックで起動（Pythonインストール不要）

## 方法2: Pythonスクリプトを直接実行する場合（開発・テスト用）

1. 依存ライブラリをインストールします。

   ```bash
   pip install -r requirements.txt
   ```

2. GUIアプリケーションを起動します。

   ```bash
   cd src
   python shapefile_normalizer_gui.py
   ```

## 方法3: EXEをビルドする場合

1. ビルドスクリプトを実行します。
   | ビルドスクリプト | 概要 |
   | ------------------------ | -------------------------------------------------------------------------------- |
   | scripts\build_exe.bat | PyInstallerで単一EXEファイルをビルド（依存ライブラリは自動インストールされます） |
2. ビルド成果物は `builds\dist\` ディレクトリに出力されます。

## アプリケーションの使用方法

3. 各タブの操作方法
   | 手順           | 内容                       | 詳細                                                                                                                                                                                           |
   | -------------- | -------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | 実行タブ       | 入力ファイル選択と処理実行 | .shpまたは.zipファイルを複数選択し、出力先ディレクトリを指定します。各ファイルのマッピング定義、EPSG、文字コードを設定後、「実行」ボタンをクリックします。                                     |
   | 設定タブ       | デフォルト設定             | バッファ距離（最小辺の長さチェック）、デフォルトの変換元EPSG、変換先EPSG、文字コードを設定します。設定の保存・読み込みが可能です。                                                             |
   | カラム設定タブ | カラムマッピング定義       | サンプルファイルから入力カラム情報を読み込み、出力カラムに対するマッピング方法（カラム代入、四則演算、カラム抽出、固定値、ファイル名、ランダム値、シーケンス値、条件分岐、ファイル名分岐等）を設定します。 |

4. カラムマッピング方式一覧
   | マッピング方式     | 概要                                           |
   | ------------------| ---------------------------------------------- |
   | カラム代入 ※       | 入力カラムをそのまま出力カラムに代入           |
   | カラム四則演算 ※   | 入力カラムに対して四則演算を実行               |
   | 複数カラム四則演算 ※ | 複数の入力カラム間で四則演算を実行             |
   | 複数カラム抽出  ※  | 複数の入力カラム間で最大値・最小値を抽出   　　|
   | 固定値             | 固定値を設定                                   |
   | ファイル名         | インポートしたファイル名（.shp）を自動設定     |
   | ランダム値         | 指定範囲のランダム値を生成                     |
   | シーケンス値       | 連番を生成                                     |
   | 条件分岐           | 入力カラムの値に応じて出力値を分岐             |
   | ファイル名分岐     | ファイル名に応じて出力値を分岐（拡張子省略可） |

   **※ 補完機能**: 指定した入力カラムのNULL値・空文字列を補完
   - **固定値**: 固定値を代入
   - **カラム条件分岐**: 指定した他カラムの値をもとに出力値を代入
   - **ファイル名分岐**: インポートしたファイル名をもとに出力値を代入
